Добавлен `asyncio.Lock` для синхронизации доступа к `last_forwarded.txt`, чтобы один обработчик проверял/сохранял пост одновременно. Используется словарь `_processing_posts` для отслеживания обрабатываемых постов. Двойная проверка предотвращает дубли, а `finally` гарантирует очистку `_processing_posts`.
